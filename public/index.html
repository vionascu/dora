Data Pipeline

How this dashboard works:

  1. INPUT: Repositories defined in ReposInput.md
  2. COLLECTION: Git and CI artifacts extracted (git_artifacts/, ci_artifacts/)
  3. CALCULATION: Raw data processed into metrics (calculations/)
  4. PRESENTATION: This dashboard displays calculated metrics

To add projects: Edit ReposInput.md with new repository URLs and run the pipeline.
